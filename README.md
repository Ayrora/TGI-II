No momento, esse modelo está usando o Mistral e está com currículos pré montados no arquivo data.py. No momento, ainda não estamos com uma base de dados grande de curriculos/candidatos

Siga os passos para rodar o programa no seu PC:

1 - Instalar o modelo de IA no Ollama
2 - Colocar o modelo para rodar (ex: ollama run mistral)
3 - Criar um novo projeto no seu decodificador (recomendo o pycharm)
4 - Colocar os arquivos aqui do github no seu projeto
5 - Rodar pip install -r requirements.txt no terminal do decodificador
6 - Rodar o main.py
